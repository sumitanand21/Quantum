Impact message
SAP Customer Code:
·
Customer Name change in Trace has an impact on where the invoice is raised
because of the Trace----> SAP Linkage .
·
Project creation and Resource Management is linked to Customer Name.
·
SAP customer code cannot be changed for Amendments, as it should match
with it's parent order SAP customer code.
Project Start Date:
·
Start date of an Order cannot be changed. Please create a Negative
Amendment for the original order and a Positive Amendment with the new
details.
Project End Date:
·
If the approved order duration is more than 12 months, then a change in end
date will lead to change in ACV. Therefore change in End date cannot be
allowed if the original approved order duration is more than 12 months.
Negative amendment for the initial order needs to be created and positive
amendment based on new dates has to be created.
·
If the approved order period is less than 12 months and the change in original
end date causes the new duration to be less than 12 months, the project
duration is affected but the ACV is not impacted.
·
If the original order period was less than 12 months and change in the End date
leads to a duration of more than 12 months, then the ACV is impacted and a
negative amendment for the original order booked and positive amendment
with new details has to created.
Vertical Sales Owner:
·
Changing the Vertical Sales Owner has an impact on the incentive that will
be paid.
Advisor:
·
Changing the Advisor has an impact on the incentive that will be paid.
Authorization type:
·
Please upload the new documents with the correct dates. (Link for same
should be provided so that he can directly go ahead and do it.)
·
If the new document has a signed date which does not belong to the same
financial year of the signed date, then it is not allowed. Please book a negative
amendment for the original order and positive amendment for the new details.
·
If the new start date is different from the original Start date then it is not
allowed. Please book a negative amendment for the original order and a
positive amendment with new details.
·
If the new end date leads to a change in duration which is more
than 12 months, then it is not allowed. Please book a negative amendment for the
original order and a positive amendment with new details.
·
If the original period was less than 12 months and the new end date leads to a
duration of more than 12 months, then it is not allowed. Please book a negative
amendment for the original order and a positive amendment with new details.
